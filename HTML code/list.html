  • Ordered Lists
  • Unorderd Lists
  • Description Lists

  3. Description List

    • This list is used to create the list of items with discription.

    • Tags Used are:dl dt and dd.

    • dl tag define the Description list

    • dt tag defines the name/terms

    • dd tag define the details of element.(Gives the description of the elements)

    Examples:

    Coffee powder

    • Coffee powder contains Caffein.

    • Excessive Comsumption is Injurious for health.

    Bags

    • Of different types:leather ,Plastics etc.

    • Used to carry heavy Items
